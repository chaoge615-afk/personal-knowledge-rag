# 安装说明

## 安装方法

OpenClaw 技能安装非常简单，只需要两步：

### 1. 复制技能文件夹到 skills 目录

找到你的 OpenClaw 工作目录下的 skills 文件夹，将 `relationship-analysis-free` 或 `relationship-analysis-full` 整个文件夹复制进去。

典型的路径：
```
你的-openclaw工作目录/skills/relationship-analysis-free
```

### 2. 重启 OpenClaw 或等待自动加载

复制完成后，重启 OpenClaw 服务，技能就会被自动加载。

```bash
openclaw gateway restart
```

## 使用方法

安装完成后，当你遇到情感相关问题时，可以直接提问，技能会自动匹配相关素材给出专业建议：

- **亲密关系矛盾分析**：和伴侣吵架、冷战、沟通问题都可以问
- **脱单追求指导**：男生追求女生、女生追求男生都有系统指南
- **聊天话题**：找不到话题的时候直接问，会给你匹配对应话题
- **约会表白**：约会技巧、表白话术都有参考
